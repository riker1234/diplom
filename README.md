Веб-приложение для сравнения цен и подбора компьютерной периферии. Реализует **систему поддержки принятия решений (СППР)** — помогает пользователю выбрать подходящий товар через интерактивный опросник, агрегируя данные с Ozon, Wildberries и Ситилинка.

## Система поддержки принятия решений (СППР)

Ключевая функциональность проекта — рекомендательный модуль, реализующий СППР без использования сторонних ML-библиотек. Алгоритм работает в три этапа:

### 1. Фильтрация (SQL)
По ответам пользователя формируется SQL-запрос, отсекающий заведомо неподходящие товары:
- **Бюджет** — исключаются товары, у которых ни в одном из магазинов нет цены в рамках бюджета
- **Тип подключения** — проводная/беспроводная для мышей и наушников, USB/XLR для микрофонов
- **Форм-фактор** — Full / TKL / компактная раскладка для клавиатур
- **Диагональ** — малый (<24"), средний (24–27"), большой (≥27") для мониторов
- **Материал поверхности** — мягкий/жёсткий для ковриков

### 2. Скоринг (Python)
Каждому прошедшему фильтр товару присваивается оценка на основе совпадений с предпочтениями пользователя. Учитываются: попадание цены в нижние 70% бюджета (+2), соответствие сценарию использования (игры, работа, стриминг, звонки), совпадение типа переключателей клавиатуры, частота обновления и тип матрицы монитора, наличие микрофона в наушниках, тип микрофона и подсветка коврика. Максимальный вклад одного критерия — 3 балла.

### 3. Сортировка и выдача
Товары сортируются по убыванию оценки, при равных оценках — по минимальной цене среди всех магазинов. Возвращаются топ-20 результатов.

## Возможности каталога

- 6 категорий: мыши, клавиатуры, мониторы, наушники, микрофоны, коврики
- Фильтрация по магазинам (Ozon, Wildberries, Ситилинк) с логикой пересечения — показываются только товары, доступные во всех выбранных магазинах одновременно
- Фильтрация по диапазону цен учитывает цены во всех магазинах
- Сортировка по цене и названию

---

## Сбор данных

Данные парсятся с трёх магазинов разными инструментами. Ozon и Wildberries используют Selenium с WebDriver — оба сайта строятся на JS и требуют реального браузера. Ситилинк отдаёт статический HTML, поэтому для него достаточно асинхронных запросов через httpx.

Для каждой категории товаров парсеры сохраняют: название, бренд, цену, характеристики, изображение и ссылку на товар. Изображения Wildberries генерируются программно по структуре CDN-серверов.

## Запуск

### Бэкенд

```bash
cd backend
pip install -r requirements.txt

# Применить миграции
alembic upgrade head

# Запустить сервер
uvicorn app.main:app --reload
```

API доступно на `http://localhost:8000`  
Swagger документация: `http://localhost:8000/docs`

### Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Основные API эндпоинты

| Эндпоинт | Описание |
|----------|----------|
| `GET /mice/` | Список мышей с фильтрами |
| `GET /keyboards/` | Список клавиатур |
| `GET /monitors/` | Список мониторов |
| `GET /headphones/` | Список наушников |
| `GET /microphones/` | Список микрофонов |
| `GET /mousepads/` | Список ковриков |
| `POST /recommendation/` | Получить рекомендации СППР по ответам опросника |

Все эндпоинты каталога поддерживают параметры `price_min` и `price_max`.
